# How To Use

## 1) Install & Setup
```
python setup.py
```

## 2) Input your API Key information in settings.py


## 3) Run to see your portfolio balance
```
python khan.py
```



### For permission errors on pip install, chown your Python lib folder
sudo chown -R $USER /Library/Python/2.7
